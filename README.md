# 🌿 Aashirwad — Premium Ecommerce Website

Aashirwad is a luxury handcrafted ecommerce website designed with a premium UI/UX experience.
It features smooth animations, modern design, and a fully functional shopping system.

---

## ✨ Features

* 🛍️ Fully functional cart system
* 💳 Multiple payment methods (COD, UPI, Card, Net Banking)
* 🎯 Admin Panel (hidden access)
* 📦 Product management via localStorage
* 🎨 Premium luxury UI design
* 📱 Fully responsive (mobile + desktop)
* 🎬 Smooth Apple-style animations
* 🖼️ Image-based product system (URL-based images)
* 🔄 Persistent data (no reset on reload)
